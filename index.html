Ajouter une image (glisser-déposer ici)

Instructions :

  • Cliquez sur la cellule que vous souhaitez modifier
  • Cliquez sur une image du bandeau pour la choisir (ou sur la première des icônes pour la supprimer)
  • À chaque modification des données, la zone de texte ci-dessous est mise à jour. Vous pouvez la copier pour la sauvegarder
  • Le bouton charger permet de charger les données de la zone de texte
  • Le bouton "Vers la barre d'adresse" permet de mettre à jour l'adresse internet avec les données du tableau, pour pouvoir le sauvegarder en favoris
  • Le bouton "Suivant identique" vous permet de vous déplacer a la cellule suivante en y copiant le même pictograme (pratique pour faire une rangée de chauffage par exemple).
